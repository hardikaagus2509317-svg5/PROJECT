1.staggered sensor reading 
	Staggered sensor reading (atau asynchronous sensor reading) adalah kondisi ketika dua atau lebih sensor membaca data pada waktu yang tidak bersamaan. Perbedaan waktu ini bisa terjadi karena perbedaan response time sensor, perbedaan sampling rate, atau keterlambatan sistem pembacaan data. Masalah utama dari staggered reading adalah data yang dihasilkan tidak mewakili kondisi yang sama pada satu waktu tertentu. Hal ini dapat menyebabkan hubungan antar parameter terlihat lemah atau tidak akurat, meskipun secara fisika sebenarnya saling berkaitan. Contohnya pada sensor suhu dan kelembapan, yang secara alami memiliki hubungan, tetapi terlihat tidak sinkron karena waktu pembacaan berbeda. Untuk mengatasi masalah ini digunakan algoritma kompensasi. Algoritma ini bertujuan untuk menyelaraskan data agar seolah-olah diambil pada waktu yang sama. Penyesuaian dapat dilakukan dengan cara menggeser data (time shift), melakukan interpolasi, atau menggunakan model matematis tertentu. Dalam jurnal yang dibahas, Pearson correlation coefficient digunakan sebagai indikator performa untuk mengukur apakah proses kompensasi berhasil. Jika nilai korelasi meningkat setelah kompensasi, berarti penyelarasan data berhasil memperbaiki dampak staggered reading. Secara keseluruhan, staggered sensor reading merupakan masalah umum dalam sistem multi-sensor, terutama pada aplikasi IoT, monitoring lingkungan, dan sistem kontrol. Oleh karena itu, teknik kompensasi sangat penting untuk memastikan data yang dihasilkan tetap akurat dan dapat dianalisis dengan benar.
	
2.moving average filter
	 Moving average filter merupakan metode pengolahan sinyal dan data untuk mengurangi fluktuasi (noise) jangka pendek, sehingga tren utama lebih terlihat jelas. Teknik ini bekerja dengan menghitung rata-rata sejumlah data terbaru, lalu memperbarui nilai rata-rata tersebut secara terus-menerus saat data baru masuk. Ada Penelitian yang membahas tentang deteksi retakan di dalam batuan dan mengukur tingkat bukaannya menggunakan data dari mesin bor. Karena data pengeboran mengandung banyak gangguan atau fluktuasi kecil, digunakan moving average filter untuk menghaluskan data agar perubahan akibat retakan dapat terlihat lebih jelas.Dalam penelitian teknik biasanya urutannya seperti ini: Ambil data mentah dari mesin, bersihkan data (pakai moving average), baru dianalisis lebih lanjut. Ketika data mesin bor masih terdapat noise maka program komputer bisa salah mendeteksi retakan, mengira noise adalah retakan, jadi moving average ini seperti membersihkan kaca dulu sebelum melihat keluar. Dalam pemrograman, moving average filter ini juga sangat penting karena dapat digunakan untuk mengurangi noise, membuat data lebih mudah di analisis, dan membantu pengambilan keputusan otomatis. Walaupun terlihat berguna segala sesuatu pasti ada kekurangannya, misalnya moving average ini memiliki kekurangan yaitu ia memperlambat respons sistem, terutama ketika window size terlalu besar, sehingga tidak cocok untuk aplikasi yang membutuhkan waktu respons cepat.
